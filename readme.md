# Final Arduino Project
This repository contain the final project of ESEO first year. Our project is a copy of Pacman who run on Arduino

## Code structure
- The Game work using determinist control system. `game.hpp` contain all game logic and state management.
- The `inputs.hpp` handle inputs transmissions to the Arduino.
- The `screen.hpp` handle the screen communication.
- The `main.ino` is the main file where everything is process.

## Requirement
- TFT Screen
- Two buttons
- 1 Joystick
